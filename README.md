# Satzung und Ordnungswerk

Dieses Repository enthält die aktuellen Entwürfe der Satzung sowie der Ordnungen des Vereins.
Die Dokumente werden hier versioniert, diskutiert und gemeinsam weiterentwickelt.

## Aktuelle Version herunterladen

[![Latest Release](https://img.shields.io/github/v/release/NarrenbundOstfildern/Satzung?label=Release)](https://github.com/NarrenbundOstfildern/Satzung/releases/latest)

Dort findet ihr:

* einzelne PDFs aller Ordnungen,
* ein Gesamtdokument,
* sowie eine Diff-PDF mit markierten Änderungen zur vorherigen Version.

## Änderungsvorschläge

Verfügbare Issue-Templates:

* Inhaltliche Änderung
* Redaktionelle Korrektur
* Neue Regelung
* Streichung

Bitte verwendet immer das passende Template.

### Wie erstelle ich ein Issue?

1. Klickt oben auf den Reiter „Issues“
2. Klickt auf „New Issue“
3. Wählt das passende Template aus
4. Füllt die Felder möglichst vollständig aus
5. Klickt unten auf „Submit new issue“

[![Änderungsvorschlag einreichen](https://img.shields.io/badge/Issue-Erstellen-blue)](https://github.com/NarrenbundOstfildern/Satzung/issues/new/choose)

### Änderungsvorschläge per Mail

* Die Mail muss exakt der Struktur eines GitHub-Issues entsprechen.
* Die Mail wird anschließend 1:1 als Issue übernommen.
* Unstrukturierte Vorschläge oder allgemeine Diskussionen per Mail werden ignoriert.


## Aufbau der Mail / des Issues
### Inhaltliche Änderung
```
## Betroffenes Dokument & Stelle
 
**Dokument:** <!-- z.B. Arbeitsdienstordnung -->
**Paragraph / Abschnitt:** <!-- z.B. §3 Abs. 1 -->

## Aktueller Stand
 
<!-- Was steht dort aktuell? Wörtliches Zitat oder kurze Zusammenfassung -->
> ...
 
## Vorgeschlagene Änderung
 
<!-- Wie soll die Regelung künftig lauten? Möglichst konkreter Formulierungsvorschlag -->
> ...
 
## Begründung
 
<!-- Warum ist diese Änderung sinnvoll oder notwendig? -->
```

### Redaktionelle Korrektur

```
## Betroffenes Dokument & Stelle

<!-- Welches Dokument, welcher Paragraph oder Abschnitt? -->
**Dokument:** <!-- z.B. Satzung -->
**Paragraph / Abschnitt:** <!-- z.B. §5 Abs. 2 -->

## Korrekturvorschlag
 
<!-- Wie soll es korrekt lauten? -->
> ...

## Art des Fehlers
 
<!-- Bitte ankreuzen -->
- [ ] Rechtschreibung
- [ ] Grammatik
- [ ] Zeichensetzung
- [ ] Formatierung (z.B. Einrückung, Nummerierung)
- [ ] Sprachliche Klarheit / Lesbarkeit
```

### Neue Regelung
```
## Art des Vorschlags
 
- [ ] Neuer Paragraph in einem bestehenden Dokument
- [ ] Neuer Abschnitt innerhalb eines Paragraphen
- [ ] Neues Dokument / neue Ordnung

## Betroffenes Dokument
 
**Dokument:** <!-- z.B. Jugendordnung, oder „neu" wenn neues Dokument -->
**Einfügen nach:** <!-- z.B. nach §4, oder „am Ende" -->

## Vorgeschlagener Text
 
<!-- Möglichst konkreter Formulierungsvorschlag für den neuen Inhalt -->

## Begründung
 
<!-- Warum braucht es diese Regelung? Welches Problem löst sie? -->
```

### Streichung
```
## Betroffenes Dokument & Stelle
 
**Dokument:** <!-- z.B. Beitragsordnung -->
**Paragraph / Abschnitt:** <!-- z.B. §7 Abs. 3 -->

## Zu streichender Text
 
<!-- Wörtliches Zitat der zu streichenden Stelle -->
> ...

## Begründung
 
<!-- Warum soll diese Regelung entfallen? Ist sie überholt, widersprüchlich, überflüssig? -->
```
